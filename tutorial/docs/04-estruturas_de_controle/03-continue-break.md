# Continue e break

```python
for elemento in iteravel:
    # bloco de execução
    if truth_testing:
        continue
    # Bloco de código não executado se comando "continue" é executado. Mas fluxo inicial da iteração é reexecutada.
```  

```python
for elemento in iteravel:
    # bloco de execução
    if truth_testing:
        break
    # Bloco de código não executado se comando "continue" é executado. Fluxo do loop é encerrado.
```
