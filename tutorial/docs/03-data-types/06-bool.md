## Operations on bool types

- Truth Value Testings:
    ```python
    variable_bool1 = True
    variable_bool2 = False
    variable_truth_value_testing1 = variable_bool1 or variable_bool2 # yields to True
    variable_truth_value_testing2 = variable_bool1 and variable_bool2 # yields to False
    variable_truth_value_testing3 = not variable_bool1 # yields to False
    ``` 

- Return of Comparison Operations:
    ```python
    variable_bool1 = 2 < 6
    variable_bool2 = 2 <= 6
    variable_bool3 = 2 > 6
    variable_bool4 = 2 >= 6
    variable_bool5 = 2 == 6
    variable_bool6 = 2 != 6
    variable_bool6 = 2 is 6
    variable_bool7 = 2 is not 6
    ```
